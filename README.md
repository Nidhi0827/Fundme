# FundMe - Cloud Native DevOps Project

A crowdfunding REST API built with Java Spring Boot, 
deployed using modern DevOps practices.

## Tech Stack
- Java 21 + Spring Boot
- MySQL Database
- Docker + Kubernetes
- Jenkins CI/CD
- AWS (EC2, RDS)
- Terraform + Ansible
- Prometheus + Grafana

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/campaigns | Create a campaign |
| GET | /api/campaigns | Get all campaigns |
| POST | /api/campaigns/{id}/donate | Donate to a campaign |
